Controles do Jogo

🎮 CONTROLES DO JOGO 🚶 Personagem: W / A / S / D → Andar pelo mapa F → Entrar / sair de veículos M → Abrir mapa 🚜 Veículos: W → Acelerar S → Frear / Ré A / D → Virar E → Engatar / Desengatar implemento C → Ligar / desligar motor G → Ativar / Desativar marcha automática H → Ativar / desativar AutoDrive L → Ligar / desligar implemento (Puxar/Plataforma) O → Abastecer no posto 🌾 Trabalho no campo: Q → Interagir (Silo, encher sementes, etc) Z → Abrir celular (se tiver comprado) N → Mostrar / esconder monitor de campo ⛽ Caminhão: Q → Abastecer outros veículos (Chegue do lado deles e aperte Q) 🛒 Loja e Oficina: Loja (Icone Carrinho) → Aperte Q Oficina (Icone Chave) → Aperte Q (dentro do veículo parado) Use o celular Z para acessar a loja de qualquer lugar 💬 Multiplayer: ENTER → Digitar no chat B → Fechar / abrir Chat na tela ESC → Fechar menus / Menu de pausa

Passo a Passo da Fazenda

🌱 COMO JOGAR - PASSO A PASSO 1️⃣ Começando - Aperta a seta para o lado para ir ate o trator - Entre no TRATOR (Tecla F) - Aperte a tecla C para ligar o motor - Dirija ate a cidade com o trator (W,A,S,D) 2️⃣ Preparando a terra Você precisa preparar o solo em 3 etapas: 🚜 Arar (Plow) - Engate um arado (E) - Ligue o motor (C) e passe na terra para arar 🚜 Gradear (Harrow) - Troque para uma grade - Passe na terra arada para nivelar 🌱 Plantar (Seed) - Engate a plantadeira - Encha com sementes na loja ou casa (tecla Q) - Passe na terra preparada 3️⃣ Esperar crescer - A plantação leva um tempo para crescer - Espere até a cor mudar para indicar colheita pronta 4️⃣ Colher - Use uma COLHEITADEIRA - Passe sobre a plantação madura 5️⃣ Vender - Vá até o silo para descarregar - Use um caminhão para vender sua produção no ícone de venda 6️⃣ Melhorando sua fazenda - Compre mais veículos e novas terras - Automatize com AutoDrive (G) 7️⃣ Combustível - Abasteça no posto (O) ou com o caminhão (Q) 8️⃣ Multiplayer - Entre em salas com amigos para trabalhar juntos! 💡 DICAS IMPORTANTES: - Use o monitor de campo (N) para ver o progresso - O celular (Z) permite comprar de qualquer lugar
